# CopperStuff
Adds more uses for copper to Minecraft

This is my first Minecraft mod, so it's pretty small and basic. It's made for 1.18.1

## Features
- Copper Pickaxe: Similar to gold with a hint of iron
  - Durability of 190
  - Speed of 9
